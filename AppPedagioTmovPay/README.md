## Instalação 

Use o [git](https://git-scm.com/) para clonar este repositório

```shell
git clone git@gitlab.sotran.com.br:sotran/app-pedagio-tmov-pay.git # SSH

git clone http://gitlab.sotran.com.br/sotran/app-pedagio-tmov-pay.git # HTTP
```

## Executar projeto 

- Selecionar o enviroment que deseja executar e o projeto irá executar com base no appsettings.[enviroment].json selecionado.

- Executar projeto com `Ctrl + F5` ou em executar no Visual Studio.

## Build do projeto

- Entrar na pasta do projeto `cd AppPedagioTmovPay`

- Executar o comando `electronize build target custom "win-x86;ia32"`
